============================================================
    SOLUSDT 激进短线交易策略 v8.3
    布林带回归 + DeepSeek AI 动态风控
============================================================

目标：在币安杠杆账户上进行短线合约交易。
风格：5分钟布林带均值回归 + 5倍杠杆 + DeepSeek AI 二次过滤。

一、核心逻辑
------------------------------------------------------------
本地策略根据布林带触及、RSI转向、EMA趋势、成交量放大和MACD动量
产生做多/做空候选信号。DeepSeek AI 作为第二层过滤，对入场审批、
出场持仓/反手决策、定期持仓复评进行判断。硬风控（-1.0%最大亏损、
120分钟负收益强平）不能被AI突破。

二、技术指标及参数
------------------------------------------------------------
| 指标         | 参数          | 用途               |
|-------------|---------------|--------------------|
| EMA         | 5, 13, 30     | 短线趋势            |
| RSI         | 7             | 超买超卖判断        |
| MACD        | 8, 17, 9      | 快速动量变化        |
| 布林带(BB)   | 14, 2.0       | 波动率边界          |
| ATR         | 7             | 真实波幅            |
| 成交量均线   | 10            | 放量确认            |

时间框架：5分钟K线。MTF上下文：15分钟 + 1小时（默认仅传给AI）。

三、入场条件（本地信号质量过滤）
------------------------------------------------------------

【做多候选】
  1. ATR 占价格比例 >= 0.045%（市场活跃）
  2. 最低价触及布林带下轨（BB_lower * 1.003）
  3. 当前K线收阳（close > open）
  4. RSI 上升且 <= 55
  5. 成交量 >= VOL_MA * 1.5
  6. EMA 趋势：EMA5 > EMA13 > EMA30
  7. 布林带宽度 >= 0.6%
  8. RSI 底背离（加分项）

【做空候选】
  1. ATR >= 0.045%
  2. 最高价触及布林带上轨（BB_upper * 0.997）
  3. 当前K线收阴（close < open）
  4. RSI 下降且 >= 45
  5. 成交量 >= VOL_MA * 1.5
  6. EMA 趋势：EMA5 < EMA13 < EMA30
  7. 布林带宽度 >= 0.6%
  8. RSI 顶背离（加分项）

最近3个AI skip后进入临时严格模式，提高信号门槛。

四、DeepSeek AI 层
------------------------------------------------------------

【入场 AI】
输入：最近12根K线、指标、账户状态、MTF方向、最近交易结果。
可返回：approve（仓位10%-70%可调）或 skip。
不可：把做多改成做空、做空改成做多。

【出场 AI】
触发条件：软性出场到达、30分钟复评、AI动态阈值再次触达。
可返回：close、hold（带新TP/SL阈值）、reverse_long/reverse_short。
不可：突破 -1.0% 硬止损或 120min 负收益强平。

【AI 质量门槛】
AI approve 连续5次亏损 → AI 仓位上限临时降回基础48%。

五、出场条件
------------------------------------------------------------
硬性出场（强制执行）：
  - 浮亏 >= -1.0% → 强平
  - 持仓超120分钟仍为负收益 → 强平
  - 保证金率低于动态底线 → 强平

软性出场（AI判断）：
  - 止盈触发：>= 1.5%（或ATR动态最高3.5%）
  - 止损触发：<= -1.0%（或ATR动态最低0.6%）
  - 移动止盈：盈利达1.2%启动，回撤0.45%平仓
  - 盈利回撤保护
  - MACD/RSI 动能转弱
  - 30分钟空闲复评
  - AI 计划阈值每30分钟复查

六、仓位管理
------------------------------------------------------------
  杠杆：5倍
  基础资金使用率：48%（AI可调 10% - 70%）
  目标名义仓位：净值 * 5 * cap_use
  保证金率底线：仓位<30%→150%, <70%→138%, 否则→130%
  每次只持1笔。

七、风控规则
------------------------------------------------------------
  1. 单笔硬止损 -1.0%（AI不可豁免）
  2. 负收益持仓120分钟强平
  3. 日亏损达起始净值15% → 停到次日
  4. 连续亏损5笔 → 暂停30分钟
  5. 动态保证金率底线保护
  6. AI approve 连续亏损限制（5次→降回基础仓位）
  7. SQLite 双写持久化（JSON + SQLite）
  8. 滑点统计：信号价 vs 实际成交价

八、MTF 多周期
------------------------------------------------------------
  15分钟和1小时趋势由 EMA30 + MACD 柱判断。
  默认传给 DeepSeek 做参考。
  可选开启本地MTF过滤：大周期偏空拒做多，偏多拒做空。
  大周期震荡：基础仓位减半（48% → 24%）。

十、重要风险提示
------------------------------------------------------------
  1. 5倍杠杆下回撤也会放大，注意保证金率
  2. DeepSeek AI 是辅助工具，不保证盈利，务必设止损
  3. API密钥请关闭提现权限，仅开交易权限
  4. 中国大陆用户需代理或 Cloudflare Worker 中继访问币安API
  5. 本策略为短线激进型，不构成投资建议
