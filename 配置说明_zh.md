# 配置说明 — 从零开始运行 SOLUSDT 交易机器人

本文档面向**零基础用户**，手把手教你从无到有完成配置。

---

## 第一步：注册币安并开通 API

### 1.1 注册币安账户

访问 [binance.com](https://www.binance.com) 注册账户，完成身份认证（KYC）。


### 1.2 开通杠杆交易

进入币安网页版 → 左上角菜单 → **杠杆交易** → 开通**全仓杠杆**账户。

### 1.3 向杠杆账户转入 USDT

币安 App → 钱包 → 划转 → 从**资金账户**划转 USDT 到**杠杆账户**。

> 建议初始转入 $30~$50 USDT 进行测试。

### 1.4 创建 API 密钥

**⚠️ 密钥等于你的账户密码，绝对不能泄露！**

1. 币安网页版 → 右上角头像 → **API 管理**
2. 点击 **创建 API**
3. 选择 **系统生成**（不要选第三方）
4. 给密钥起个名字（如 "SOL_bot"）
5. **权限设置**（关键步骤）：
   - ✅ 允许现货及杠杆交易
   - ✅ 允许合约交易（读取余额用，非必需）
   - ✅ 允许钱包读取（读取资金账户，非必需）
   - ❌ **不要**勾选"允许提现"
   - ❌ **不要**勾选"允许划转"（除非你想让程序自动划转资金）
6. 完成安全验证（邮箱/短信/2FA）
7. 你会得到 **API Key** 和 **Secret Key**，**立刻复制保存**。Secret Key 关闭后不可再查看。

## 第二步：安装 Python 环境

### 2.1 下载 Python

1. 打开 [python.org](https://www.python.org/downloads/)
2. 下载最新版 Python（3.10+）
3. 安装时**务必勾选** ✅ "Add Python to PATH"
4. 点击 Install，等待完成

### 2.2 验证安装

打开**命令提示符**（Win+R → 输入 `cmd` → 回车），输入：

```bash
python --version
```

如果显示 `Python 3.x.x` 说明安装成功。

---

## 第三步：下载程序

### 3.1 下载代码

从 GitHub 下载本项目压缩包并解压，或将文件夹复制到任意位置。

### 3.2 进入项目文件夹

在命令提示符中：

```bash
cd C:\Users\你的用户名\Desktop\github
```

### 3.3 安装依赖

```bash
pip install ccxt numpy requests
```

等待安装完成（约 1~2 分钟）。

## 第四步：配置程序

### 4.1 创建配置文件

将 `config.example.json` 复制一份，重命名为 `config.json`。

### 4.2 编辑 config.json

用**记事本**打开 `config.json`，填入以下内容：

```json
{
    "api_key": "这里填你的API Key",
    "secret": "这里填你的Secret Key",
    "testnet": false,
    "proxy": "http://127.0.0.1:7890",
    "usd_cny_rate": 7.25,
    "initial_equity": 30
}
```

| 字段 | 说明 | 必填 |
|------|------|:---:|
| `api_key` | 币安 API Key | ✅ |
| `secret` | 币安 Secret Key | ✅ |
| `proxy` | 代理地址（中国大陆必填） | 视情况 |
| `usd_cny_rate` | 美元兑人民币汇率 | 否 |
| `initial_equity` | 初始本金（USD），用于计算总盈亏 | 否 |

### 4.3 配置代理（中国大陆用户）

如果你在中国大陆，币安 API (`api.binance.com`) 被墙，必须通过代理访问。

1. 打开你的代理软件（Clash/Clash Verge/v2rayN 等）
2. 查看 HTTP 代理端口号：

| 软件 | 默认端口 |
|------|---------|
| Clash / Clash Verge | 7890 或 7897 |
| v2rayN | 10809 |
| Shadowsocks | 1080 |

3. 把端口号填入 `config.json` 的 `proxy` 字段，例如：
```json
"proxy": "http://127.0.0.1:7890"
```

> 如果不用代理也能访问币安，把 `proxy` 设为空字符串 `""`。

## 第五步：运行程序

### 5.1 启动交易机器人

```bash
python MainProgramme.py
```

启动后会自动：
- 显示各账户余额
- 同步现有持仓
- 进入策略循环（每 5 分钟检查一次）

如果要清空所有持仓：
```bash
python MainProgramme.py --close-all
```

### 5.2 启动监控仪表盘

打开**另一个**命令提示符窗口：

```bash
python monitor.py
```

浏览器打开 **http://localhost:8888**，即可查看实时资产和交易记录。

## 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| `Read timed out` | 代理未开或端口不对 | 检查代理软件是否运行，端口是否匹配 |
| `Connection refused` | 代理地址错误 | 确认 `127.0.0.1:端口` 正确 |
| `Insufficient balance` | 杠杆账户余额不足 | 向杠杆账户转入更多 USDT |
| `Invalid API key` | 密钥填错或已删除 | 重新创建 API 密钥 |
| `Permission denied` | API 权限不足 | 检查是否勾选了"允许现货及杠杆交易" |
| 仪表盘空白 | 旧进程占用端口 | 先 Ctrl+C 停掉旧的，再重新运行 |
| 仪表盘数字不更新 | 数据加载中 | 等待 30~60 秒（代理加载较慢） |

## 安全提醒

1. **永远不要分享你的 API Secret Key** — 任何人拿到它都可以操作你的账户
2. **不要勾选"允许提现"** — API 密钥只需要交易权限
3. **设置 IP 白名单** — 在币安 API 管理页面限制只有你的 IP 可以调用
4. **先小资金测试** — 确认流程跑通后再增加投入
5. **定期检查日志** — 关注异常交易和错误信息
