============================================================
       SOLUSDT 量化交易机器人 — 项目成果说明
============================================================

一、项目概述
------------------------------------------------------------
基于币安交易所 API 的 SOLUSDT 自动化杠杆交易系统。
采用"趋势跟随 + 均值回归"混合策略，支持多空双向短线交易，
3倍杠杆放大收益，严格的动态风控保护本金。


二、核心成果
------------------------------------------------------------

【1. 交易机器人 (MainProgramme.py)】
  - 自动连接币安杠杆账户，每5分钟扫描K线信号
  - 技术指标：EMA(5/13/30)、RSI(7)、MACD(8/17/9)、布林带、ATR
  - 入场逻辑：趋势确认 + RSI回调 + MACD动能 + 放量确认（5条件评分）
  - 出场逻辑：硬止损(-1.5%)、ATR动态止损、移动止损、止盈(+3%)、时间止损(2h)
  - 动态仓位：根据保证金率自动缩减仓位
  - 保证金率保护：<128%不开仓、<128%强制平仓
  - 仓位越大底线越高：20%仓位→150%底线、60%→138%、90%→130%
  - 自适应阶段：净值<$50激进、$50-100滚雪球、$100+稳步、$500+保守

【2. 资产监控仪表盘 (monitor.py)】
  - HTTP服务(端口8888)，浏览器打开即可查看
  - 实时显示：总资产(USD/CNY)、杠杆账户净值、保证金率、SOL价格
  - 资产负债横向条形图（正值右=资产、负值左=做空负债）
  - 交易记录表（时间、动作、方向、数量、价格、盈亏）
  - 今日盈亏 & 总盈亏（可配置初始值）
  - SOL 5分钟K线走势图
  - 3天资产趋势折线图（双轴USD/CNY）
  - 资产明细表（四账户：资金/现货/杠杆/合约）
  - 策略参数实时展示
  - 10秒自动刷新，Chart.js 可视化

【3. 策略文档】
  - SOLUSDT_Trading_Strategy.md：完整策略参数和规则
  - 策略说明-白话版.md：新手友好的策略解释


三、技术栈
------------------------------------------------------------
  - Python 3.14
  - ccxt（币安API封装）
  - numpy（技术指标计算）
  - Chart.js（前端图表）
  - HTTP Server（内置，无额外依赖）


四、使用方法
------------------------------------------------------------
  1. pip install ccxt numpy requests
  2. 复制 config.example.json 为 config.json，填入API密钥
  3. 中国大陆用户配置代理 proxy 字段
  4. python MainProgramme.py          # 启动交易
  5. python monitor.py    # 启动仪表盘 → http://localhost:8888
  6. python MainProgramme.py --close-all  # 紧急平仓
